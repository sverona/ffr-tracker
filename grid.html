flexbox playground

this is a little page that lets you play with the Flexbox layout.

i made this in a couple afternoons because flex property names and values were too many and nonintuitive to me and the existing guides were too detailed or too flashy.

set container or item properties using the dropdowns.

flex-flow shorthand for {flex-direction} {flex-wrap}.
flex-direction: ; direction in which to order the flex items.
flex-wrap: ; whether to wrap the flex items or force them onto one line.

wrap-reverse acts perpendicular to flex-direction; e.g., if flex-direction: column, flex-wrap: wrap-reverse reverses the horizontal order of the columns.

flex shorthand for {flex-basis} {flex-grow} {flex-shrink}.
flex-basis: ; item width before growing or shrinking.

a width value, or one of "fill", "max-content", "min-content", "fit-content", "content."

flex-grow: ; factor by which to grow item relative to others.

more precisely, if some items have nonzero values for flex-grow and there is extra space, items are assigned a percentage of the extra space proportional to their flex-grow value.

flex-shrink: ; factor by which to shrink item relative to others.

more precisely, if some items have nonzero values for flex-shrink and there is not enough space, items are shrunk by a percentage of the surplus proportional to their flex-shrink value.

order: ; ordinal placement of item.

defaults to 0. thus, order: 1 will place item after unordered items, whereas order: -1 will place item before unordered items.
